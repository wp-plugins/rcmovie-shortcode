Embed videos from rcmovie.de using shortcode rcmovie

== Description ==

Simple to include videos from the german rcmovie.de platform using a shortcode on a wordpress page.

= Features: =

simple include the player on your page

= Documentation =

**Basic example:**

[rcmovie id="57e83990249ebddbb8cc"]

This shortcode will embed the rcmovie.de player. 

== Installation ==

1. Upload `rcmovie-shortcode` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode on your page or `<?php echo do_shortcode('[rcmovie id=videoidfromrcmovie]'); ?>` in your templates
